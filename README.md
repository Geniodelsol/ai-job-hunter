# AI Job Hunter

AI-powered job search assistant for marketing and project management professionals.

## Goal

Automatically:

- monitor career pages
- collect vacancies
- rank jobs by profile match
- generate tailored CVs
- generate cover letters
- track applications
- analyze job market trends
